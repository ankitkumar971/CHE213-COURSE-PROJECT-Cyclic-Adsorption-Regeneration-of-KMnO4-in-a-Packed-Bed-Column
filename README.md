Project Overview

This project investigates the cyclic adsorption–regeneration behavior of KMnO₄ in a packed bed column using activated charcoal as the adsorbent.

The study evaluates breakthrough characteristics, adsorption efficiency, regeneration performance, and bed utilization through experimental observations and MATLAB-based data analysis.

Objectives

- Study adsorption behavior in a packed bed
- Analyze regeneration effects on adsorption performance
- Construct breakthrough curves
- Calculate Length of Unused Bed (LUB)
- Evaluate adsorption efficiency
- Compare fresh and regenerated beds
